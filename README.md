## 🫁 Lung Capacity Prediction (Machine Learning Project)

## 📌 Project Overview
This project analyzes and predicts **lung capacity (LungCap)** based on various factors such as age, height, smoking habits, gender, and birth method.

The goal is to understand how these features influence lung capacity and build a machine learning model for prediction.

---

## 📊 Dataset Information

The dataset contains **725 records** with the following features:

| Feature     | Description |
|------------|------------|
| LungCap    | Lung capacity (target variable) |
| Age        | Age of the individual |
| Height     | Height in inches |
| Smoke      | Smoking status (yes/no) |
| Gender     | Male/Female |
| Caesarean  | Born via Caesarean (yes/no) |

---

## 🎯 Objectives
- Perform data cleaning and preprocessing
- Conduct exploratory data analysis (EDA)
- Visualize relationships between variables
- Build a machine learning model to predict lung capacity
- Evaluate model performance
